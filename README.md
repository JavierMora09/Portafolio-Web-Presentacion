# Porfolio Profesional - Francisco Javier Mora Serrano

Este repositorio contiene el código fuente de mi porfolio personal. Es una plataforma diseñada para mostrar mi trayectoria como **Ingeniero de Sistemas e Informática**, mis proyectos destacados y las tecnologías que domino.

El sitio enfoca una experiencia de usuario fluida, con un diseño moderno basado en **Glassmorphism** y componentes altamente interactivos.

## 🚀 Tecnologías Utilizadas

*   **React**: Biblioteca principal para la creación de interfaces de usuario.
*   **TypeScript**: Para un desarrollo seguro con tipado estático.
*   **Tailwind CSS**: Framework de CSS para un diseño rápido y responsivo.
*   **Lucide React**: Biblioteca de iconos minimalistas.
*   **Swiper.js**: Para el manejo de carruseles interactivos en la sección de proyectos.
*   **Vite**: Herramienta de construcción para un entorno de desarrollo ultra rápido.

## ✨ Características Principales

*   **Diseño Responsivo**: Totalmente adaptado para dispositivos móviles, tablets y ordenadores de escritorio.
*   **Sección Hero**: Presentación personal con acceso directo al CV y redes sociales.
*   **Proyectos Dinámicos**: Galería de proyectos con descripciones, stacks tecnológicos y enlaces a código/demo.
*   **Sobre Mí**: Detalle de mi perfil profesional y áreas de especialización.
*   **Formulario de Contacto**: Sección dedicada para facilitar la comunicación directa.

## 🛠️ Instalación y Configuración Local

Si deseas clonar este proyecto y ejecutarlo en tu máquina local, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/JavierMora09/Porfolio_Web.git
    ```

2.  **Entra al directorio del proyecto:**
    ```bash
    cd Porfolio_Web
    ```

3.  **Instala las dependencias:**
    ```bash
    npm install
    ```

4.  **Ejecuta el proyecto en modo desarrollo:**
    ```bash
    npm run dev
    ```
